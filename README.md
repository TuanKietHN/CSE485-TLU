# CSE485-TLU

## Công nghệ Web

### **BTTH/BTTH01/Câu 1:**
#### **Hướng dẫn - User Guide:**
1. **Tải code về**, tải **XAMPP** hoặc cài extension **PHP Serve** để chạy project.
2. **Truy cập trang index hiển thị sản phẩm**:
   - **Dùng PHP Serve**:
     - Mở dự án bằng IDE, vào đường dẫn `CSE485-TLU/BTTH/BTTH01/Cau 1/index.php`.
     - Chuột phải vào file code chọn **PHP Serve: Serve Project**.
     - Extension sẽ tự động điều hướng trình duyệt đến URL:  
       `http://localhost:3000/CSE485-TLU/BTTH/BTTH01/Cau%201/index.php`.  
       Đây là trang hiển thị sản phẩm cho người dùng không có quyền quản trị.
   - **Dùng XAMPP**:
     - Đưa toàn bộ dự án vào thư mục `htdocs`.
     - Chạy (start) **Module Apache**.
     - Truy cập trình duyệt với URL:  
       `http://localhost/tlu/CSE485-TLU/BTTH/BTTH01/Cau%201/`.  
       Đây là trang hiển thị sản phẩm cho người dùng không có quyền quản trị.
3. **Truy cập trang admin/index hiển thị giao diện CRUD sản phẩm**:
   - **Dùng PHP Serve**:
     - Mở dự án bằng IDE, vào đường dẫn `CSE485-TLU/BTTH/BTTH01/Cau 1/admin/index.php`.
     - Chuột phải vào file code chọn **PHP Serve: Serve Project**.

       Đây là trang giao diện CRUD sản phẩm cho người dùng có quyền quản trị.
   - **Dùng XAMPP**:
     - Đưa toàn bộ dự án vào thư mục `htdocs`.
     - Chạy (start) **Module Apache**.
     - Truy cập trình duyệt với URL:
